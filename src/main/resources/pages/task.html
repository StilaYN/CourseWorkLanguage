Константы в Java

Константа — это специальная переменная, значение которой устанавливается при ее объявлении и не может быть изменено в последующем ходе выполнения программы.

Для описания констант в языке Java используется ключевое слово final.

Для описания целочисленных констант в языке Java используется тип int.

Формат записи: final int имя_константы=значение;

Примеры:

  1. Целая положительная десятичная константа без явного указания знака – любое десятичное целое число без знака:
    final int M=1234;
  2. Целая положительная десятичная константа с явным указанием знака – любое десятичное целое число со знаком “+”:
    final int E=+1234;
  3. Целая отрицательная десятичная константа с явным указанием знака – любое десятичное целое число со знаком “-”:
    final int E=-1234;

Верные записи констант согласно автоматной грамматике G[]:

  • final int a=1234;
  • final int b12=-1234;
  • final int c4=+1234;
